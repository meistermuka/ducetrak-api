# Ducetrak-API

## Description

[Ducetrak](https://trak.muka.io) API repository. Based on the
[Nest](https://github.com/nestjs/nest) framework, this API will serve as the main
backend for the Ducetrak app.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
